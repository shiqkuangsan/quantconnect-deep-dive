## 怎么读

两套文档定位不同，配合使用：

| 系列 | 定位 | 核心问题 | 适合场景 |
|------|------|---------|---------|
| [**架构参考手册**](./quantconnect-deep-dive/README.md) | 想明白 | 为什么这样设计？ | 理解原理、技术决策参考 |
| [**团队上手指南**](./quantconnect-hands-on/README.md) | 跟着做 | 怎么跑起来？怎么改？ | 动手实操、团队 onboarding |

**推荐学习路径：** 先跟着 hands-on 系列动手跑起来，遇到"为什么"的问题时回 deep-dive 查原理。

### 团队 4 周上手计划

| 周 | hands-on 文档 | 目标产出 |
|----|-------------|---------|
| 第 1 周 | 01 → 02 → 03 | 能写策略、看懂回测报告、完成 5 个实战 lab |
| 第 2 周 | 04 → 05 | 本地环境跑通、掌握 Research 数据探索 |
| 第 3 周 | 06 → 07 → 08 | 自定义指标 → 数据源 → 券商插件，具备二开能力 |
| 第 4 周 | 09 → 10 | 策略上线 + 团队协作规范落地 |

## 关于 QuantConnect

[QuantConnect](https://www.quantconnect.com/) 是一个开源量化交易平台，核心是 [LEAN 引擎](https://github.com/QuantConnect/Lean)（16,000+ GitHub stars）。支持 Python / C# 策略开发，覆盖美股、期权、期货、外汇、加密货币等 7 大资产类别，对接 20+ 券商。
